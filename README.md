# 🎬 CineBoard — Production Bible

> **Workshop Edition** — This version was built live during a workshop using **Vibe Coding** directly in **GitHub Copilot Agent Mode**.

CineBoard is an interactive web application that serves as a digital "production bible" for audiovisual productions. It gives each department its own organized space with tasks, documents, comments, and real-time activity.

## Origin

This app was born from a real need: Sofía, a 21-year-old film student and assistant director, needed a tool to organize pre-production across her projects. Everything was scattered across Google Drive, WhatsApp, and random documents. CineBoard solves that with a centralized, accessible space for the entire crew.

## 🛠️ Vibe Coding + Agent Mode

This version was built **entirely with Vibe Coding** using GitHub Copilot in Agent Mode:

- **No code written manually** — everything was generated by the agent from the stakeholder's requirements
- **From zero to a working app** in a single workshop session
- The agent set up the project, chose the stack, created the data models, designed the components, and resolved build errors autonomously

## Features

- **Dashboard** with overall stats and per-department progress
- **8 departments** — Photography, Art Direction, Cinematography, Sound, Production, Costume Design, Makeup & Hair, Editing
- **Department workspace** with Checklist, Documents, and Team tabs
- **Interactive tasks** — mark as completed, expand details, add comments
- **Create new tasks** per department
- **Community feed** — real-time activity across the entire production
- **Documents view** — mood boards, spreadsheets, links, and images organized by department
- **Responsive sidebar** with mobile navigation

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4 + shadcn/ui
- Zustand (state management)
- Lucide React (icons)
